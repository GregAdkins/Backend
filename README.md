# 5thSocial Backend

## Developer Setup

1. Copy `.env.example` to `.env` and update your settings.
2. Run `npm install`
3. Start with `npm run dev`

## Includes

- `/src` — app entrypoint
- `/tests` — unit test stubs
- `/migrations` — SQL schema setup
- `/config` — preserved from original
- `.env` and `.env.example`
- Docker + Postman setup
- Swagger API docs
